Trabalhando em CSS

FONT-FAMILY

Usar a família (Conjunto de três fontes), ao invés de uma só.

FONT-SIZE

Medidas Absolutas
cm, mm, in, px, ponto(pt)

Medidas Relativas
em(relativa ao M maísculo de uma letra), ex(relativa ao tamanho x), rem (root), vw, vh(view-height), %

Os mais recomendados são EM e PX

16px = 1em

FONT-WEIGHT

Lighter, normal, bold, bolder

FONT-STYLE

Normal, italic, oblique, initial, inherit
Aqui muda apenas a aparência, não a importância do texto

TEXT-DECORATION

Line-through, overline, underline

SHORT-HAND FONT

Ao invés de usar o padrão: 

            
    font-family: Arial, Helvetica, sans-serif; 
    font-size: 1em;
    font-weight: normal; 
    font-style: oblique; 
    text-decoration: underline;
Tem a posibilidade de usar em uma só linha, mas nessa ordem exata:
font-style -> font-weight -> font-size -> font-family

IMPORT

Também tem a possibilidade de importar uma letra a partir do google fonts (ou outros), é só colocar o código em style e chamar onde quer a fonte depois.

Também tem o FONT FACE, quando por ex o cliente te entrega o arquivo da fonte que vai ser usada ou quando vc baixa o arquivo da fonte em um site. Pesquisa pq nao entendi direito. 

        
        @font-face {
        font-family:*nomedafamily*; 
        scr: url(pasta/'nomedafonte.tipo') format('opentype'),  
        url ('pasta/nomedoarquivo.outrotipo') format('truetype'); }

TEXT-INDENT

Dá pra colocar um espaço antes do ínicio do parágrafo.
